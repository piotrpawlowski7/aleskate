# Ale Skate Club — Website Summary

**Ale Skate Club** is a non-professional, community-driven skateboarding club based in Warsaw, Poland. The website serves as the club's digital hub — providing information about the community, session schedules, skate spot listings, how to join, and answers to frequently asked questions.

## Purpose

The site promotes an inclusive skateboarding community that welcomes everyone regardless of age, skill level, or equipment. The club's name plays on the Polish word "Ale" (meaning "But"), conveying the message: *"But wait — you CAN skate."*

## Key Messaging

- **Inclusivity first:** No experience needed, no board required, all ages and levels welcome.
- **Community-driven:** Built on friendliness, encouragement, and shared passion for skating.
- **Free and accessible:** No membership fees for outdoor sessions; multiple locations across Warsaw.
- **Consistent schedule:** Regular sessions on Tuesdays, Thursdays, and Saturdays.
- **Year-round activity:** Outdoor sessions in warmer months, indoor parks (AvePark, Woodpark) in winter.

## Website Structure

| Section       | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| **Hero**      | Bold intro with club name, tagline, and calls to action (Join / See Schedule). |
| **About**     | Club story, philosophy, and key stats (3× weekly, no board required, Warsaw-based, infinite welcome). |
| **Schedule**  | Breakdown of Tuesday (flagship), Thursday (mid-week), and Saturday (weekend) sessions. |
| **Spots**     | Six Warsaw skate locations: AvePark, Woodpark, Świętokrzyski Bridge Skatepark, Pole Mokotowskie, Shaber Bowl, Służewiec Skatepark. |
| **Join**      | Four-step onboarding: Follow on Instagram → Check the spot → Show up → Be part of it. |
| **FAQ**       | Seven questions covering gear, experience level, fees, locations, attire, winter sessions, and kids. |
| **Instagram** | Social media call-to-action linking to @aleskate.club. |
| **Footer**    | Session summary, spot list, social links, and copyright. |

## Technology

Single-page website built with HTML, CSS, and vanilla JavaScript. Deployed via GitHub Pages. Uses Google Fonts (Bebas Neue, DM Mono, Space Grotesk) and features a dark theme with yellow (#f5e642) and orange (#ff6a1a) accent colors.
